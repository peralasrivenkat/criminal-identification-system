# FaceNet Model Folder

Place optional pretrained FaceNet files here when real FaceNet inference is required.

Supported options:

```text
models/facenet_model.pb
```

or TensorFlow checkpoint files inside:

```text
models/facenet/
```

Pretrained model files are not included in GitHub because they can be large and may have separate licensing requirements.
